GitHub的ssh配置

GitHub 为了安全性考虑，在2020年7月就准备对所有需要使用身份认证的git命令切换成基于令牌的身份认证。并且从2021年8月13日开始，在 GitHub.com上对Git操作进行身份验证时将不再接受帐户密码。

GitHub设置

复制公钥内容到剪切板，登录GitHub帐号，点击右上角头像——Settings—— SSH and GPG keys ——[New SSH key]

起个名字，粘贴刚才复制的公钥保存。这样就设置好了。
